Usage
-----
Just add the dependency in your project:

_Maven_
```xml
<dependency>
    <groupId>io.github.orpolyzos</groupId>
    <artifactId>spring-web-initializr</artifactId>
    <version>3.0.0</version>
</dependency>
```

Contributing
